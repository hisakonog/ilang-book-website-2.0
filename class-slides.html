These are some of our lecture slides. Click the thumbnail picture to see and download the PDF of the slideshow.

You can also download the slides as TeX files and use them as a base for your own slides. TeX files are used with LaTex, a free typesetting program. The TeX file of each slideshow comes in a zip file with accompanying image files. Please don't forget to change the path for the embedded image files in each TeX document.
